# Tech Stack

## Core Technologies
- **Language:** [TypeScript](https://www.typescriptlang.org/) (v5.6.2+) - For robust type-safety and modern JavaScript features.
- **Runtime:** Obsidian environment (Plugin API).

## Frontend & UI
- **Framework:** [React](https://reactjs.org/) (v18.3.1) - For building dynamic and responsive UI components.
- **State Management:** [Zustand](https://github.com/pmndrs/zustand) (v5.0.0+) - A small, fast, and scalable bearbones state-management solution.
- **Data Table:** [TanStack Table](https://tanstack.com/table/v8) (v8.20.5+) - Headless UI for building powerful tables and data grids.
- **Icons:** [Lucide React](https://lucide.dev/) (v0.446.0+) - Consistent icon set compatible with Obsidian.

## Build & Tooling
- **Bundler:** [esbuild](https://esbuild.github.io/) (v0.24.0+) - Extremely fast JavaScript bundler.
- **Type Checker:** [TypeScript Compiler (tsc)](https://www.typescriptlang.org/docs/handbook/compiler-options.html) - For static analysis and build verification.

## Testing
- **Test Runner:** [Vitest](https://vitest.dev/) (v2.1.1+) - Next-generation testing framework with native ESM support.
- **Coverage:** [@vitest/coverage-v8](https://vitest.dev/guide/coverage.html) - For comprehensive code coverage analysis.
